## 📋 工作流程

```
抓取热点 → AI 生成叙事短片选题 → AI 写三幕剧剧本
         → AI 生成角色设定卡（通义万相绘画提示词）
         → AI 生成场景九宫格卡
         → AI 生成可灵提示词包（按场景分组）
         → 打包 Word 文档拍摄指南
         → 你用通义万相生成参考图 → 可灵生成视频 → 剪映成片 → 发布B站
```

## 🎯 角色和场景一致性解决方案

### 三层锁定机制

1. **角色锁定**：生成主角正面半身像 → 上传可灵"主体参考" → 所有镜头复用同一参考图（一致性>96%）
2. **场景锁定**：生成场景九宫格（9个视角）→ 锁死空间结构 → 避免不同镜头场景跳变
3. **多镜头连续生成**：同一场景的3-4个镜头一次性生成 → 可灵自动保持连续性

### 可灵3.0 关键特性
- 主体参考：上传角色参考图，所有镜头保持角色一致
- 多镜头连续生成：一次生成3-6个连贯镜头
- 每日66免费灵感值（约6次生成）
